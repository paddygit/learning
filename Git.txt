Problem faced:
Was receiving permission related messages

Cause:
SSH was not added to the GitHub website

Steps:
$ cat ~/.ssh/id_rsa.pub
This is your SSH key. Add it to your SSH keys in the repository.
In gitlab go to
profile settings -> SSH Keys
